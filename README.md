# Geospatial Healthcare Analysis Dashboard

## Overview
This repository features a geospatial analytics project focused on location-based healthcare intelligence. It includes a synthetic dataset containing geographical coordinates/regions and an exported dashboard demonstrating mapping workflows and regional healthcare metrics.

## Project Assets

* **`Geospatial_Healthcare_Dashboard.pdf`:** A static export of the geospatial dashboard. This visualization demonstrates how mapping can be used to identify regional trends, facility distribution, or patient access to care.
* **`Synthetic-Geospatial-Healthcare-Dataset.csv`:** The underlying synthetic dataset containing the location-based data used to plot the map visualizations.

## Potential Use Cases
Geospatial data in healthcare is critical for:
* Analyzing patient distance to care facilities.
* Mapping the spread of regional health conditions.
* Optimizing supply chain and resource allocation across different hospital branches.

## How to Use
1. **View the Dashboard:** Open the `Geospatial_Healthcare_Dashboard.pdf` to see the final spatial visualizations and map layers.
2. **Explore the Data:** The CSV dataset can be imported into standard BI tools (like Tableau or Power BI), GIS software (like QGIS or ArcGIS), or Python libraries (like Folium or GeoPandas) to practice building custom map layers and spatial queries.

> **Disclaimer:** The dataset provided in this repository is completely synthetic and generated for educational and portfolio demonstration purposes. It does not contain any real demographic or Protected Health Information (PHI).
